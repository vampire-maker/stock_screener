# 快刀手晚进早出策略 v2.0 更新说明

## 更新日期
2025-12-26

## 策略概述

基于326只股票的实盘数据回测，优化了选股条件，提高胜率和收益率。

## 策略变化对比

### 参数调整

| 参数 | v1.0 (原版) | v2.0 (优化版) | 说明 |
|------|-------------|---------------|------|
| 涨幅区间 | 2.5%-5.0% | 2.8%-4.5% | 收窄范围，避开弱势股和过度追高 |
| 量比 | >1.0 | 1.0-1.6 | 收窄上限，避开量比过高(诱多) |
| 换手率 | 无 | U型筛选 | 新增：换手率<2.5% 或 >18% |
| 近20日涨停 | 是 | 是(优先) | 保持不变，作为加分项 |
| 市值 | ≤200亿 | ≤200亿 | 保持不变 |

### 新增参数

```python
# 换手率U型分布参数
'TURNOVER_MODE': 'U_SHAPE',           # U型模式
'TURNOVER_LOW_MAX': 2.5,              # 低换手率上限 (主力控盘)
'TURNOVER_HIGH_MIN': 18.0,            # 高换手率下限 (资金活跃)
'TURNOVER_AVOID_MIN': 5.0,            # 避开区间下限
'TURNOVER_AVOID_MAX': 10.0,           # 避开区间上限 (最差区间)

# 量比收窄
'MAX_VOLUME_RATIO': 1.6,              # 量比最大值
```

## 回测验证结果

### 策略对比 (326只股票样本)

| 策略 | 股票数 | 平均涨幅 | 胜率 | 涨停数 |
|------|--------|----------|------|--------|
| 原策略 | 326 | -0.86% | 16.0% | 7 |
| +近20日涨停 | 57 | -0.67% | 24.6% | 3 |
| 量比1.0-1.5 | 111 | -0.54% | 18.9% | 3 |
| +换手率<2% | 39 | -0.30% | 23.1% | 1 |
| **+换手率>20%** | 17 | **+0.27%** | **41.2%** | 2 |
| **综合策略+近20日涨停** | 6 | **+3.44%** | **66.7%** | 1 |

### 关键发现

1. **换手率是最强预测指标**，呈现U型分布
   - 换手率>20%: 平均涨幅+0.27%, 胜率41.2%
   - 换手率<2%: 胜率23.1%
   - 换手率5-10%: 最差区间，平均涨幅-1.28%

2. **近20日涨停**是稳定的加分项
   - 胜率提升8.6%
   - 涨停组中32.5%有近20日涨停历史

3. **量比适中偏低更好**
   - 量比1.0-1.5优于量比>2.0
   - 量比过高可能是诱多陷阱

## 使用建议

### 选股时间
- 交易日14:30

### 优先级排序
1. **S级**: 近20日有涨停 + (换手率<2.5% 或 >18%) → 重点跟踪
2. **A级**: 换手率优 → 正常关注
3. **B级**: 其他 → 观察名单

### 次日操作
- **高开>2%**: 持有，冲高卖出
- **平开/低开**: 立即止损
